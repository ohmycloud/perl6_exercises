We can now nest formatting codes inside other formatting codes.

You can use multiple angle brackets as your delimiters. Angle-bracket delimited code can contain stuff with the same number of angle brackets or fewer, but not more.

So if we wanted to show some hash access code, these would be legal:

say $foo<bar>

Here, the separation between Pod and code is easier to see. say $foo<bar>
